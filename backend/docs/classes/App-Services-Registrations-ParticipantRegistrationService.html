ParticipantRegistrationService

  • Application

ParticipantRegistrationService.php : 18

Service handling registration operations from the participant's perspective.

It provides methods for participants to register for events, pay, cancel, and retrieve their tickets. Access is strictly restricted to users with the ROLE_PARTICIPANT role.
