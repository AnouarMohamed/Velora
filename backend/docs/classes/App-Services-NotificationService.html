NotificationService
in package
  • Application

NotificationService.php : 19

Service responsible for managing and dispatching application-level notifications.

This service centralizes all notification logic, ensuring consistent messaging across the platform for different user roles (Admins, Organizers, Clients, and Participants). It handles both internal system notifications and external-facing updates.
